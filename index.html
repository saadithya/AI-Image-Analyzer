Privacy-first Chrome extension to detect AI-generated images and save images with AI-generated descriptions. All analysis is performed locally.

Overview

AI Image Analyzer helps you evaluate images quickly and privately. Use the right-click menu on any image in Chrome to:

  • Check for AI generation — produces a confidence score, concise reasoning, and indicators.
  • Save with description — download the image with an AI-generated caption that includes objects, scene type, and mood.
  • Local-first processing — image analysis runs in your browser; no uploads to external servers.

Technical summary

The extension uses a combination of pixel-level heuristics (approximate ELA, noise variance) and a small client-side model where available. It leverages Chrome's activeTab permission and context menu user gestures to access images securely.

Permissions: storage, downloads, contextMenus, scripting, notifications, activeTab
Host access: temporary per-user gesture (activeTab) — no broad host permissions

How to use

  1. Install the extension from the Chrome Web Store.
  2. Right-click any image and choose "Check for AI Generation" or "Save Image with AI Description".
  3. Open the extension popup to view local history and change settings (resize, include date).

Security & privacy highlights

  • No telemetry or analytics.
  • All image analysis runs locally; nothing is uploaded to remote servers.
  • Minimal permissions — uses activeTab to access images only after an explicit user action.
