Privacy Policy

Last updated: February 2025

AI Image Analyzer ("we", "our", "the Extension") is committed to safeguarding your privacy. This Privacy Policy explains how the extension operates, what information it accesses, and how that information is handled.

Summary

  • No personal data collected — The extension does not collect, transmit, or store personal data.
  • Local-only image processing — Images are analyzed directly in your browser; no image is uploaded to any server.
  • Minimal permissions — We only use permissions essential to provide the extension features.

Information the Extension Accesses

  • Image Data: When you right-click an image and select an action, the extension fetches that image for local processing. The image is not transmitted elsewhere.
  • Local Settings: Preferences such as resize options and filename formatting are stored using Chrome's storage API.
  • Local Activity History: A short history of your recent analyses (up to 50 entries) is stored locally so you can review previous actions through the extension popup.

Permissions Used

storage — Save user settings and local history.
downloads — Save AI-described images to your device.
contextMenus — Provide right-click options on images.
scripting — Inject small content scripts for pixel-level analysis.
notifications — Display results and completion messages.
activeTab — Temporary access to the tab after a user gesture.

Data Sharing & Transmission

We do not send any data to external servers. No analytics, no tracking, no remote processing. All logic runs inside your browser using Web Workers and optional Chrome on-device AI APIs.

Third-Party Code Resources

The extension may load client-side libraries such as TensorFlow.js for local inference. These libraries are static code files and do not collect or send user data. You may alternatively bundle these files locally if self-hosting.

Your Rights & Controls

  • You can clear your local history anytime using the "Clear History" button.
  • You can disable or uninstall the extension at any time.
  • The extension performs no external storage, so no data export requests are applicable.

Security Measures

The extension follows privacy-by-design principles: minimized permissions, no remote data transfer, and complete local execution. Files downloaded to your device follow your system
