Privacy Policy

Last updated:

Thank you for using AI Image Analyzer (the “Extension”). We built this Extension with a strong commitment to privacy. This policy explains what data (if any) we collect, why we collect it, how it is stored, and the controls you have.

Summary

  • No personal data collection: The Extension does not collect personal information or telemetry.
  • Local processing: Image analysis and caption generation occur on your device—no images or content are uploaded to our servers.
  • Permissions: The Extension requests only minimal permissions required for functionality. See details below.

What data the Extension accesses

The extension accesses only the image the user chooses to analyze (via right-click context menu). It also stores small user settings and a local history of recent actions in the browser storage. Specific items:

  • Image data — The image you right-click is fetched for local analysis and/or for packaging into a downloaded file if you choose “Save with AI Description.” This is processed locally and not sent anywhere.
  • Local settings — Settings such as image resize dimensions and whether to prefix filenames with the date are stored locally using Chrome chrome.storage.
  • History — A short local history of recent image checks and saves (max 50 entries) is stored in local storage so the popup can show recent activity. This history is stored only on your device.

Permissions & why they are required

storage — Store extension settings and local history (local device only)
downloads — Save image files to the user's Downloads folder
contextMenus — Add right-click menu entries for images
scripting — Inject a small content script when required for pixel-level analysis
notifications — Notify users when analyses complete or files are saved
host_permissions — Limited to common image file types (jpg, png, webp, jpeg) to fetch the image the user selects

Do we transmit data off your device?

No. The Extension does not upload images, content, or usage data to external servers. All analysis is performed locally using the browser's capabilities (Chrome AI APIs where available and a local Web Worker). We do not collect analytics or crash reports.

Third-party services

The extension loads a publicly-hosted TensorFlow.js model for client-side inference (if available) and may load TFJS runtime from a CDN inside the worker to perform local analysis. These requests are only used to fetch necessary code/models and are not tied to your identity. If you prefer, you can host the TFJS resources locally in the extension package to avoid CDN requests.

Your choices and rights

  • You can clear the local history from the extension popup ("Clear History").
  • You can remove or disable the extension at any time via Chrome's Extensions page.
  • Because no personal data is collected, there are no data export or deletion requests related to external servers.

Security

We follow best practices to minimize risks: minimal permissions, local-only processing, and no remote storage of user images. However, please note that downloaded files written to your filesystem are subject to your operating system's security and permissions.

Children

The Extension is not designed for children under 13. We do not knowingly collect information from children.

Changes to this privacy policy

We may update this policy from time to time. When we do, we will update the "Last updated" date at the top of this page.
